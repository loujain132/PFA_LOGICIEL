---

## 📌 Description du Projet

Ce projet permet d’automatiser le traitement des ventes à partir de fichiers CSV / Excel.

- 📊 Analyse des données  
- ⚙️ Calcul automatique des indicateurs  
- 📈 Génération des graphes  
- 🌐 Intégration avec Django  

---

## 🎯 Objectif

- Automatiser les calculs  
- Réduire les erreurs humaines  
- Faciliter l’analyse  
- Transformer les données en décisions  

---

## 🧠 Technologies utilisées

- Python  
- Django  
- Pandas  
- Matplotlib  
- Openpyxl
